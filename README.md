# ECOLOGICALhome - Premium Eco-Residencies

Interaktivna prezentacija luksuznih, energetski nezavisnih objekata. Fokus projekta je na modernom dizajnu i vrhunskom korisničkom iskustvu kroz napredne animacije.

## 🚀 Tehnologije
* **Frontend:** HTML5, CSS3 (Custom Variables), JavaScript (ES6+)
* **Animacije:** [GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/) - ScrollTrigger & Timeline
* **Scroll:** [Lenis](https://github.com/darkroomengineering/lenis) - Smooth Scroll rešenje
* **Fontovi:** Montserrat & Inter (Google Fonts)

## ✨ Ključne funkcionalnosti
* **Custom Preloader:** Personalizovana animacija učitavanja.
* **Smooth Scrolling:** Tečno kretanje kroz sadržaj bez trzanja.
* **Horizontal Storytelling:** Unikatna sekcija sa horizontalnim skrolovanjem za prikaz procesa rada.
* **Responsive Design:** Potpuna optimizacija za mobilne uređaje i tablete.
* **Interactive UI:** 3D tilt efekti na karticama i interaktivna FAQ sekcija.

## 🛠️ Instalacija
Sajt je statičan i ne zahteva dodatne instalacije. Dovoljno je otvoriti `index.html` u bilo kom modernom pretraživaču.
